# Svelte Snake Workshop

Welcome to the Svelte Snake Workshop!

## Get started

Install the dependencies …

```bash
npm install
```

… then start the [Svelte kit development server](https://kit.svelte.dev/):

```bash
npm run dev -- --open
```
